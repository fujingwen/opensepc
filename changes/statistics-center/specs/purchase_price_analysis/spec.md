# 采购价格分析规格

## ADDED Requirements

### Requirement: 采购价格分析列表查询

系统 SHALL 提供采购价格分析分页列表，并支持按产品和时间范围筛选。

#### Scenario: 按条件查询采购价格分析列表

- **Given** 系统中存在项目建材产品数据
- **When** 用户按产品类别、产品名称、产品规格、质量监督机构、进场时间范围、填报时间范围查询
- **Then** 系统返回符合条件的分页结果
- **Then** 每条结果至少包含产品类别、产品名称、产品规格、产品数量、产品价格、工程名称、质量监督机构、进场时间、填报时间

#### Scenario: 首屏不满足前置条件时不查询

- **Given** 用户首次进入采购价格分析页面
- **When** 用户尚未选择产品类别、产品名称、产品规格
- **Then** 系统不自动查询采购价格分析列表

#### Scenario: 必填条件校验

- **Given** 用户位于采购价格分析页面
- **When** 用户未选择产品类别、产品名称或产品规格就点击查询
- **Then** 系统提示这三个条件为必选项

### Requirement: 采购价格分析展示口径

系统 SHALL 按与建材产品页面一致的产品口径展示产品类别、产品名称和产品规格。

#### Scenario: 产品规格展示名称

- **Given** 采购价格分析查询条件中选择了产品规格 ID
- **When** 列表返回采购价格分析结果
- **Then** 表格中的产品规格展示规格名称，而不是规格 ID

#### Scenario: 导出按钮位置对齐原型

- **Given** 用户查看采购价格分析页面
- **When** 页面渲染工具栏
- **Then** 导出按钮应位于工具栏区域，并与统计分析页面位置保持一致

### Requirement: 采购价格分析汇总

系统 SHALL 在采购价格分析列表最后一行展示平均价格汇总。

#### Scenario: 展示平均价格

- **Given** 用户完成采购价格分析查询
- **When** 表格渲染合计行
- **Then** 系统不展示总数量和总金额
- **Then** 系统展示平均价格

#### Scenario: 平均价格格式

- **Given** 系统已计算平均价格
- **When** 页面展示合计行
- **Then** 平均价格保留两位小数
