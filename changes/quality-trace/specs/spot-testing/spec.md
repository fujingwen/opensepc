# 抽测缺陷建材产品 - 功能规格

## 1. 查询功能

### 查询条件

| 字段 | 类型 | 说明 |
|------|------|------|
| 产品名称 | input | 对应检测产品名称，支持模糊查询 |
| 生产批号 | input | 对应生产批号，支持模糊查询 |
| 生产厂家 | input | 对应生产厂家，支持模糊查询 |
| 检测项目 | input | 对应检测项目名称，支持模糊查询 |
| 检测日期 | daterange | 精确到年月日，范围查询 |
| 有无对比数据 | dict | 字典项：有(1)、无(2) |

### 查询行为

- 用户输入查询条件后点击"查询"按钮
- 系统根据条件查询数据并返回列表
- 点击"重置"按钮清空所有查询条件

## 2. 列表功能

### 表格字段

| 序号 | 列名 | 数据来源 | 格式 |
|------|------|----------|------|
| 1 | 检验检测工程名称 | project_name | 文本 |
| 2 | 检测项目 | check_project_name | 文本 |
| 3 | 检测产品名称 | product_name | 文本 |
| 4 | 生产厂家 | factory_name | 文本 |
| 5 | 生产批号 | batch | 文本 |
| 6 | 检测日期 | check_time | yyyy-MM-dd |
| 7 | 报告日期 | report_time | yyyy-MM-dd |
| 8 | 检验参数 | check_project_name | 文本 |
| 9 | 结论 | conclusion | 字典 |
| 10 | 有无对比数据 | has_check_data | 字典：有/无 |
| 11 | 操作 | - | 删除按钮 |

### 操作行为

- 支持分页显示
- 支持排序功能
- 操作列显示删除按钮

## 3. 删除功能

### 删除行为

- 点击删除按钮弹出确认框
- 确认后执行逻辑删除（设置del_flag=1）
- 删除成功刷新列表

## 4. 导入功能

### 导入字段

| 字段名 | 对应数据库字段 | 说明 |
|--------|---------------|------|
| 项目名称 | project_name | 工程名称 |
| 材料名称 | product_name | 产品名称 |
| 生产批号 | batch | 批号 |
| 生产厂家 | factory_name | 生产厂家 |
| 检测项目 | check_project_name | 检测项目 |
| 检测日期 | check_time | 检测时间 |
| 检测机构 | check_organize | 检测单位 |
| 报告日期 | report_time | 报告时间 |
| 不合格参数 | conclusion | 结论 |

### 导入行为

- 点击导入按钮弹出导入对话框
- 支持Excel文件上传
- 解析Excel数据并批量插入
- 导入成功后刷新列表
