# 缺陷建材厂家规格

## ADDED Requirements

### Requirement: 缺陷建材厂家支持分页查询

系统 SHALL 支持按检验检测生产厂家、产品名称、生产批号分页查询缺陷建材厂家数据。

#### Scenario: 查询缺陷建材厂家

- **WHEN** 用户输入检验检测生产厂家、产品名称或生产批号进行查询
- **THEN** 系统应返回分页结果

### Requirement: 缺陷建材厂家列表字段

系统 SHALL 在列表中展示厂家比对所需字段。

#### Scenario: 展示厂家列表

- **THEN** 列表应展示建材平台生产厂家、检验检测生产厂家、产品名称、生产批号、检测项目、检测日期
