# 未解决问题导航

本目录统一存放基于 `openspec/changes`、前后端代码、数据库和原型手册交叉核对后的未解决问题清单。

## 当前文件

- `未解决问题总览.md`：按系统业务流程归并后的总问题清单，是本目录主入口。
- `统计中心历史遗留问题.md`：统计中心相关的历史独立遗留清单，保留为补充记录。

## 收录范围

1. `tasks.md` 中仍未完成，且经代码或数据库核对后仍成立的事项。
2. `issues.md` 中已确认但暂不处理、待验证或待确认的事项。
3. 对照《系统操作流程总览-阅读版.md》《操作手册对照-业务流程与差异总览.md》后确认的流程偏差。
4. 提案、代码、数据库三者之间口径不一致，尚未完成回写收口的事项。
