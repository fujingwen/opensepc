# 基础数据与账号权限

## 1. 模块范围

本模块对应迁移清单中的以下功能：

- 基础数据-生产企业
- 基础数据-代理商
- 基础数据-施工企业
- 组织、部门、岗位管理
- 角色与权限管理

- 账号信息及登录-账号信息维护
- 账号登录功能

## 3. 迁移映射建议

### 3.1 系统表映射

建议继续沿用当前已经确认的映射关系：

- `base_user -> sys_user`
- `base_organize -> sys_dept`
- `base_position -> sys_post`
- `base_module -> sys_menu`
- 角色授权相关 -> `sys_role`、`sys_user_role`、`sys_role_menu`
- `base_dictionarytype -> sys_dict_type`
- `base_dictionarydata -> sys_dict_data`
- `base_province -> base_region`

### 3.2 企业主数据承载

企业主数据建议继续以 `t_companyinfo` 为唯一业务主表，不再拆出新的生产企业表、代理商表、施工企业表。页面和服务层按企业类型、用户类型、角色做视图隔离。
