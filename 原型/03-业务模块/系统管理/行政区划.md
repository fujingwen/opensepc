# 行政区划表字段规范化

## 修改说明

将 `master.base_province` 表中的字段从 `F_` 前缀格式修改为小写加下划线的命名格式，与其他表（如 `sys_user`）保持一致。

## 新旧字段对应关系

| 旧字段名 (F_前缀) | 新字段名 (小写+下划线) |
|-----------------|---------------------|
| F_Id | id |
| F_ParentId | parent_id |
| F_EnCode | en_code |
| F_FullName | full_name |
| F_QuickQuery | quick_query |
| F_Type | type |
| F_Description | description |
| F_SortCode | sort_code |
| F_EnabledMark | enabled_mark |
| F_CreatorTime | create_time |
| F_CreatorUserId | create_by |
| F_LastModifyTime | update_time |
| F_LastModifyUserId | update_by |
| F_DeleteMark | delete_mark |
| F_DeleteTime | delete_time |
| F_DeleteUserId | delete_user_id |

## 无需修改的字段

以下字段已经是正确格式：

- tenant_id
- create_dept
- del_flag

## 执行时间

修改日期：2026-03-20

## 注意事项

修改字段名后，请确保对应的业务代码、视图、存储过程、触发器等也同步更新字段名称，否则可能会出现报错。
